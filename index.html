Air KOALAty Report

A place to find the air quality and weather of cities made with Angular 7, Express, MongoDB, and Auth0.

Authorization

Auth0 is used for authorization to let users save their favorite cities to their list.

Angular Material

The card, button, theme, and navbar components are Angular Material.

Api

The api is from Air Visual and it provides the weather and air quality index for different cities.

Back end

The back end uses Express, Mongoose, and MongoDB to send requests to the API and to update the user's list.
